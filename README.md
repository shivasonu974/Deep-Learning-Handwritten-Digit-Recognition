# Task 2: Handwritten Digit Recognition

## Objective

Build a deep learning model to recognize handwritten digits using the MNIST dataset.

## Environment Setup

Install dependencies:

```bash
pip install tensorflow numpy matplotlib
```

## Files

* task2_digit_recognition.ipynb
* task2_report.pdf
* task2_report.docx
* README.md
* screenshots/

## How to Run

1. Open notebook in Jupyter or Google Colab
2. Run all cells sequentially
3. Observe training and evaluation outputs

## Model Summary

* Input: 28×28 grayscale image
* Hidden Layer: 128 neurons
* Output: 10 classes (digits 0–9)

## Results

* Training Accuracy: 90.93%
* Test Accuracy: 91.94%
* Prediction Example: 7 → 7

## Conclusion

The neural network successfully learned to classify handwritten digits with high accuracy.
